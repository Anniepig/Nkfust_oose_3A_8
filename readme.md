使用案例描述	使用者可透過平台取得更優惠的票價或活動贈品。
主要參與者	開發人員、遊客。
利害關係人與目標	主辦單位：提供優惠的展覽資訊。

主要成功情節	參與者	系統
	1.	進入活動及優惠專區，
觀看展覽名稱、抽獎規則、抽獎獎品。

2.	使用者如果要參加抽獎則點選”我要抽獎”

3.	使用者如獲得優惠票價及活動贈品時，則出現恭喜您得獎對話框並跳出”得獎者聯絡資料”區塊。

4.	得獎者輸入”得獎者姓名”、”得獎者連絡電話”及”得獎者聯絡信箱”後，案送出。

6.	網頁顯示”得獎代碼”，使用者到信箱查看，會收到一組得獎代碼。

7.	當天展覽時再以此代碼兌換優惠票券或贈品。	1.	進入”活動及優惠專區”的頁面。

3.	要求使用者輸入的”得獎者姓名”、”得獎者連絡電話”及”得獎者聯絡信箱”。

4.1系統檢查輸入資料格式是否正確，有錯時，系統要求重新輸入。

4.	將使用者填妥的資料回傳至系統內。

5.	系統顯示使用者剛填入的資料，使用者核對後如正確則跳往7，錯誤則跳要求重新填寫得獎聯絡資料。

6.	得獎代碼顯示於介面中並自動發送得獎代碼至使用者信箱中。


例外情節	如果網站伺服器無法運作，則轉至404錯誤頁面。
